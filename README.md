## Instructions
1. Download the data and add it to `res/` directory. Your directory tree should look like this:
```
context-free-network/
  src/
  res/
    train2017/
    test2017/
    val2017/
    annotations/
```

2. `cd` into `src/`
3. Run `make`
4. Run `jupyter notebook`
